A collection of **datasets and data exploration notebooks** used across research and teaching projects.

## 📖 About

This repository serves as a central data store for various datasets and exploratory notebooks used in research, education, and data journalism projects by Amr Eleraqi.

## 📂 Contents

- Curated datasets (CSV, JSON, Excel formats)
- Exploratory notebooks for each dataset
- Data dictionaries and documentation
- Preprocessing scripts

## 💡 How to Use

Each dataset folder contains:
1. The raw data file
2. A Jupyter notebook with initial exploration
3. A README explaining the data source and variables
